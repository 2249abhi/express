# express
Step1: run setup commands:

a) npm init -y
b) npm i express
c) npm i mongoose
d) npm install -g nodemon
e) npm i hbs 
f) add .gitignore

Step2: create index.js file and run cmd: nodemon index.js

Step3: to run with hbs
cmd: nodemon index.js -e js,hbs

Step4: install middleware
cmd: npm i body-parser
